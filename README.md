### 升级pip

```shell
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple --upgrade pip
```

### pip换源

```shell
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```

### 安装backtrader

```shell
pip install backtrader[plotting]
```

### AKShare数据源
```shell
pip install akshare
```
